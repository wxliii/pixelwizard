# PixelWizard Homepage

Static research project homepage for:

**PixelWizard: Towards Efficient High-Fidelity Video Generation at Ultra-Large Spatial Resolution**
## Edit Content

- `index.html`: title, authors, abstract, method, results, media placeholders, contact, and links.
- `script.js`: BibTeX copy behavior.
- `styles.css`: layout, colors, responsive behavior.
- `assets/hero-teaser.svg`: temporary teaser placeholder. Replace it when the real figure is ready.

## Add Paper Assets

Suggested locations:

- `assets/pixelwizard.pdf`
- `assets/teaser.png`
- `assets/videos/*.mp4`

After adding those files, update the links and media blocks in `index.html`.

## Preview

Open `index.html` directly in a browser. There is no build step and no dependency install.

## Deploy On GitHub Pages

1. Push the repository to GitHub.
2. In the GitHub repository, open `Settings` > `Pages`.
3. Set the source to the `main` branch and `/root`.
4. The page will be available at `https://wxliii.github.io/pixelwizard/`.
